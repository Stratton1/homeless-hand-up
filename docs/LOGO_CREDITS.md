# Retailer Logo Credits

This project stores retailer logos locally in `public/logos/retailers/`.

## Source Repositories
- Primary source (full-colour SVG files): `detain/svg-logos`
  - https://github.com/detain/svg-logos
- Fallback source (when needed): `simple-icons/simple-icons`
  - https://github.com/simple-icons/simple-icons
